# 04 - 新标签页打开帖子并评估正文与评论

## 目标

在每个帖子的独立标签页中，完整评估正文和评论，判断它是否算有效帖，并提取购买决策信息。

## 输入

- 候选帖列表或初步有效帖列表
- 时间范围过滤条件，可能为空
- IP 范围过滤条件，可能为空

## 动作

1. 对每个候选帖，使用“在新标签页中打开链接”的方式打开。
2. 不要复用搜索结果页覆盖打开帖子。
3. 不要关闭已经打开的帖子标签页。
4. 每个帖子打开后至少停留 5 秒，不要快速扫过。
5. 在这 5 秒内完成初步判断：时间/IP是否符合要求、是否疑似广告、正文是否有购买参考价值。
6. 读取帖子 URL、发布时间或修改时间、IP 地址。
7. 读取正文，并记录正文里和购买相关的信息。
8. 在同一标签页中读取评论区。
9. 遇到“展开”“展开更多”“展开 N 条回复”等折叠入口，都要展开后再阅读。
10. 每个帖子至少查看 5 条评论；如果评论不足 5 条，查看全部可见评论。
11. 评论较多时最多查看 50 条评论。
12. 优先查看点赞量高、回复数多、楼中楼多、争议明显的评论。
13. 直到达到评论数量上限、没有新的有用评论加载，或到达底部。
14. 综合正文和评论，最终判断是否算有效帖。

## 正文重点看什么

- 价格
- 版本/型号
- 电池/续航
- 配件/加装
- 骑行体验
- 做工/品控
- 颜值/舒适度
- 规则/上牌/合规信息
- 广告或引流痕迹
- 正文是否有实际体验、价格、缺点、对比或避坑信息

## 评论重点看什么

- 对价格有帮助的评论
- 对续航有帮助的评论
- 对品控、异响、刹车、轮胎、安全有帮助的评论
- 对服务、上牌、改装、配件有帮助的评论
- 对真实使用体验有帮助的评论
- 对广告、软广、门店套路、价格水分有帮助的评论
- 高赞或高回复且能代表争议点的评论

## 排除这些内容

- 纯“同问”
- 只有表情或简短起哄
- 与购买无关的闲聊
- 重复而没有新增信息的评论
- 纯情绪发言
- 点赞数本身

## 有效性判断

帖子只有满足以下条件才算有效帖：

- 不是广告贴、推广贴、软广或明显带货引流帖
- 发布时间/修改时间符合用户指定范围，若用户指定了时间范围
- IP 符合用户指定范围，若用户指定了 IP 范围
- 正文或评论至少一方有购买参考价值
- 已查看必要评论数量，或评论不足时已查看全部可见评论

如果正文和评论都没有购买参考价值，该帖子不计入有效帖。

## 产出

- 每篇有效帖的唯一 URL
- 每篇有效帖的发布时间/修改时间和 IP
- 每篇有效帖的正文摘要
- 每篇有效帖的正面评论要点
- 每篇有效帖的负面评论要点
- 每篇有效帖的中性/参考评论要点
- 已查看评论数量
- 是否出现高赞/高回复争议点
- 跳过帖子原因，例如广告、时间不符、IP不符、信息不足、无购买参考价值
- 页面限制说明，例如评论折叠、加载失败、登录或验证码阻塞

## 完成标准

- 每篇候选帖都已用独立新标签页打开并保留。
- 每篇帖子至少停留 5 秒后再做有效性判断。
- 已尽量展开折叠评论和折叠回复。
- 已查看 5-50 条评论，或在评论不足时查看全部可见评论。
- 已优先覆盖高赞、高回复评论。
- 已区分正文观点和评论观点。
- 已给出有效帖或跳过原因。
